천태종 서울 구강사의 1페이지 정적 홈페이지입니다. GitHub Pages 배포를 목적으로 제작되었습니다.

---

## 소개

서울 강동구 천호동에 위치한 천태종 구강사의 공식 홈페이지입니다.  
소개, 찾아오시는 길, 법회 일정, 연락처를 한 페이지에 담은 단순한 정적 사이트입니다.

## 로컬에서 보는 방법

**방법 1 – 직접 열기**  
`index.html` 파일을 더블클릭하면 기본 브라우저에서 바로 열립니다.

**방법 2 – VS Code Live Server (권장)**  
1. VS Code에서 프로젝트 폴더를 열기
2. `index.html` 우클릭 → *Open with Live Server* 선택
3. 파일 저장 시 브라우저가 자동 새로고침됩니다

---

## GitHub Pages 배포 방법

1. GitHub에 저장소 생성 후 전체 파일을 `main` 브랜치에 push
2. 저장소 페이지에서 **Settings → Pages** 이동
3. *Branch* 항목에서 `main` 브랜치, 폴더는 `/ (root)` 선택 후 **Save**
4. 잠시 후 `https://<username>.github.io/<repository>/` 주소로 접속 가능
